Справочник «Меню»

Назначение

 

Предназначен для заполнения и печати меню. Меню – это список блюд, создаваемый для покупателя. Блюда в меню можно свободно добавлять и удалять, поддерживая, таким образом, актуальность меню для покупателя. Набор используемых в информационной базе меню определяется в справочнике Виды меню.

 

Реквизиты

 

  • Группа. Группа элементов справочника Меню, к которой принадлежит данное меню.
  • Наименование. Наименование меню.  
  • Товар. Элемент номенклатуры, соответствующий позиции меню, по которой определяются параметры при печати меню. Ссылается на справочник Номенклатура.  
  • Единица измерения. Единица измерения позиции меню. Ссылается на справочник Упаковка и единицы измерения. 
  • Номер в группе. Число, соответствующее положению элемента меню в группе.
  • Запрещён вывод этого элемента при печати меню. Признак временного запрета продажи позиции меню. 
  • Рецептура. Рецептура, соответствующая позиции элемента номенклатуры в меню. Ссылается на документ Рецептуры.
  • Выбор картинки. Картинка кнопки меню во фронте. 

Дополнительная информация

 

Заполнение справочника возможно через кнопку Подбор или через Обработку:

  • Заполнить – форма автозаполнения справочника.
  • Очистить – удаляет все элементы справочника.
  • Выгрузить – форма выгрузки прайс-листов.  
  • Обновить все группы - если у группы заполнен параметр Группа номенклатуры, то данные группы заполняются всеми элементами, точно так же как в справочнике Номенклатура (Пример,  есть группа Завтрак, у которой Группа номенклатуры - Блюда. При исполнении этой команды группа Завтрак будет заполнена такими же позициями как в группе Блюда в номенклатуре). 
  • Заполнить наименования позиций - заполняет  наименования позиций по параметру Товар.
  • Заполнить наименования групп - заполняет наименования групп по параметру Группа номенклатуры.
  • Скопировать структуру групп - копирует всю структуру групп, как в справочнике Номенклатура не заполняя её элементами.
  • Заполнить по выпускам блюд – форма заполнения справочника по табличной части «Товары»  документов Выпуск блюд.

 

Меню кнопки Печать позволяет сформировать печатную форму справочника. При выборе пункта Печать меню с параметрами открывается диалоговое окно настройки параметров печати меню.

 

 

Примечание. В справочнике есть возможность интерактивного удаления элементов по кнопке Удалить позицию меню.
