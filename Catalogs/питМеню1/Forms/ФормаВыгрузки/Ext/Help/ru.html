Форма выгрузки прайс-листов справочника "Меню"

Назначение

Обработка предназначена для выгрузки прайс-листа (меню) в файл «*.xls», «*.txt» или «*.dbf», или источник данных.

Реквизиты

  • Вид прайс-листа – вид выгружаемого прайс-листа (меню). Ссылается на справочник «Виды меню».
  • Тип цен – тип цен для выгружаемого меню. Ссылается на справочник «Типы цен номенклатуры».
  • Валюта – валюта, в которой будут указаны цены в меню.  
  • Кодировка – выбор кодировки для файла.
  • Размещение – выбор размещения меню: в файл или в источник данных.
  • Подключение – путь к файлу или источнику данных, в который будет выгружаться меню.
  • Разделитель – знак разделителя, который будет использоваться в файлах с форматом «*.txt». Доступно при значении Файл реквизита Размещение.
  • Со строки/ по строку – диапазон выводимых строк из меню. Доступно при значении Файл реквизита Размещение.
  • Имя таблицы – наименование таблицы источника данных. Доступно при значении Источник данных реквизита Размещение.
