Форма автозаполнения справочника "Меню"

Назначение

Предназначена для заполнения справочника «Меню» по указанным параметрам.  

 

Способ заполнения

 

Варианты автоматического заполнения меню:

  • По складским остаткам товаров
  • По товарам поставщиков
  • По всей номенклатуре
  • По существующему меню

 

Способ создания групп

Номенклатура в меню может быть сгруппирована по видам товаров, по группам товаров либо без создания групп.

Параметры заполнения по остаткам

Если выбран способ заполнения по складским остаткам, то указываются следующие параметры:

  • Дата, на которую берутся складские остатки
  • Организация, с которой считываются остатки
  • Склад компании, с которого выбираются товары

 

Параметры заполнения по товарам поставщика

Если выбран способ заполнения по товарам поставщиков, то необходимо указать поставщика товара, в противном случае меню будет заполнено по всем поставщикам.

 

Параметры заполнения по номенклатуре

 

Здесь необходимо указать группу номенклатуры, в противном случае меню будет заполнено по всей номенклатуре.

 

Параметры заполнения по меню

 

Поле Меню доступно, если выбран способ заполнения по существующему меню. Здесь необходимо указать другой вид меню.

 

Очищать меню перед заполнением - если флажок установлен, то меню перед заполнением будет очищено.
