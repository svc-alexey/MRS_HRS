# Отчет об исправлении версионирования объектов

**Дата:** 23.03.2026  
**Задача:** Исправление логики удаления версий объектов  
**Модуль:** `CommonModules/ВерсионированиеОбъектов/Ext/Module.bsl`  
**Функция:** `ГраницыУдаленияОбъектов()`

---

## Описание проблемы

В функции `ГраницыУдаленияОбъектов()` была обнаружена критическая логическая ошибка:
- **Симптом**: Все версии объектов удалялись через месяц, независимо от настроенных сроков хранения
- **Причина**: Неправильное вычисление неверсионируемых типов - использовалась переменная `СписокТипов` только из последней итерации цикла вместо сбора всех типов из результата запроса

## Логика исправления

### Было (НЕПРАВИЛЬНО):
1. Запрос получает типы объектов с настройками версионирования
2. Для каждой группы по срокам хранения создается строка в таблице результата
3. ❌ **ОШИБКА:** Переменная `СписокТипов` после цикла содержит только типы из последней итерации
4. ❌ При вычислении "неверсионируемых" типов почти все типы попадают в эту категорию
5. ❌ Всем этим типам устанавливается срок хранения 1 месяц

### Стало (ПРАВИЛЬНО):
1. Запрос получает типы объектов с настройками версионирования
2. Для каждой группы по срокам хранения создается строка в таблице результата
3. ✅ **ИСПРАВЛЕНО:** Собираются ВСЕ типы из таблицы результата в массив `ВсеВерсионируемыеТипы`
4. ✅ При вычислении "неверсионируемых" типов используются все версионируемые типы
5. ✅ Только типы БЕЗ настроек получают месячный срок хранения

## Ожидаемый результат

После исправления версии объектов будут удаляться в соответствии с настроенными сроками:

| Срок хранения | Граница удаления | Поведение |
|--------------|------------------|-----------|
| **Бессрочно** | `'000101010000'` | Версии НЕ удаляются |
| **За последний год** | `ТекДата - 12 мес` | Удаляются версии старше года |
| **За последние 6 месяцев** | `ТекДата - 6 мес` | Удаляются версии старше 6 мес |
| **За последние 3 месяца** | `ТекДата - 3 мес` | Удаляются версии старше 3 мес |
| **За последний месяц** | `ТекДата - 1 мес` | Удаляются версии старше месяца |
| **Без настроек** | `ТекДата - 1 мес` | Удаляются версии старше месяца |
